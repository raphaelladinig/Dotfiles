## Requirements

- git
- GNU stow
- CaskaydiaCove Nerdfont
- Bibata Modern Cursor

## Installation

First, check out the dotfiles repo

```
$ git clone git@github.com/dreamsofautonomy/dotfiles.git
$ cd dotfiles
```

then use GNU stow to create symlinks

```
$ stow .
```
