## Requirements

- git
- GNU stow
- CaskaydiaCove Nerdfont
- Bibata Modern Classic Cursor

## Installation

First, check out the dotfiles repo

```
$ git clone https://github.com/raphaelladinig/Dotfiles.git
$ cd Dotfiles
```

then use GNU stow to create symlinks

```
$ stow .
```
